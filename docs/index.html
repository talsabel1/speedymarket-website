Come shop with us!

SpeedyMarket® is the ultimate solution for a streamlined and efficient grocery store experience for customers, managers and suppliers alike.

Say goodbye to long lines and complicated store navigation!

  • End-to-end shopping: manage lists, navigate, scan, and pay
  • Easy inventory management and order tracking
  • Streamlined supply chain from supplier to customer

Participating supermarkets use a QR code to be scanned on entrance to manage the shopping experience. Customers scan barcodes to add items to cart. The SpeedyMarket® application serves as a portal for all three types of users: customer, manager, supplier.

Services

SpeedyMarket® is powered by cutting-edge cloud and AI technologies for seamless shopping and optimized performance

Azure SQL Database

Reliable and scalable database management for secure data storage and fast retrieval.

SignalR

Real-time communication to synchronize user interactions across devices, for up-to-date inventory management. When an item is added to a cart, inventory and out-of-stock items are updated in real-time and reflected on all devices.

Azure Cloud Functions

Http triggered functions providing serverless computing for efficient execution of backend operations and business logic.

Computer Vision

Leveraging advanced image-to-text capabilities to convert handwritten shopping lists into digital format, allowing users to upload written lists directly into the app.

OpenAI API

Utilizing natural language processing to extract ingredients from online recipe URLs and automatically add them to the user's shopping list.

Google API Integration

Integration with Google Maps for locating nearby supermarkets and Google Pay for secure, streamlined payment processing, enhancing the overall shopping experience.
